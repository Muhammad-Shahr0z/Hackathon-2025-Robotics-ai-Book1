# Module 5: Capstone Projects

**Focus**: Building complete autonomous systems.

---

## Overview

Apply everything you've learned to build a complete autonomous humanoid system that integrates perception, planning, and control.

---

## Capstone Project

### The Autonomous Humanoid

Build a complete autonomous humanoid system that integrates perception, planning, and control.

**Requirements**:
- Voice command recognition
- Path planning and navigation
- Object detection and manipulation
- Full system integration

**Evaluation**: Demonstrate the robot completing a complex task autonomously.
