# Quickstart: GitHub Pages Image Path Fix

## Setup

1. Ensure the robologo.png file is in the `static/img/` directory
2. Update all references to the image to use the correct path format
3. Test locally before deploying to GitHub Pages

## Testing Locally

```bash
# Start local development server
npm run start

# Build and test production version locally
npm run build
npm run serve
```

## Deployment

1. Commit all changes
2. Push to main branch
3. GitHub Pages will automatically deploy the updated site
4. Verify the image displays correctly on the deployed site
