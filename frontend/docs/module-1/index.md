# Module 1: The Robotic Nervous System (ROS 2)

**Focus**: Middleware for robot control.

---

## Overview

This module introduces ROS 2 (Robot Operating System 2), the middleware that serves as the nervous system for modern robots. You will learn how to build distributed robot systems using nodes, topics, and services, and how to bridge Python AI agents to ROS controllers.

---

## What You Will Learn

- ROS 2 Nodes, Topics, and Services
- Bridging Python Agents to ROS controllers using rclpy
- Understanding URDF (Unified Robot Description Format) for humanoids
- Visualizing and simulating robots with RViz and Gazebo

---

## Chapters

### Chapter 1: Introduction to Embodied AI and Robotics

Explore the fundamental concepts of embodied intelligence and the role of robotics in the physical world.

### Chapter 2: ROS 2 Fundamentals: Nodes, Topics, and Services

Master the core architecture of ROS 2, the middleware that powers modern robotics.

### Chapter 3: ROS 2 Tools: RViz, Gazebo, and the CLI

Learn the essential tools for visualizing, simulating, and debugging robot systems.

---

- Basic Python programming
- Linux/Ubuntu familiarity (recommended)
- Understanding of basic robotics concepts
