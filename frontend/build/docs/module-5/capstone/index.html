Capstone Project: The Autonomous Humanoid

Description: Build a complete autonomous humanoid system that integrates perception, planning, and control.

Project Requirements​

Voice Command Recognition​

  • Integrate OpenAI Whisper for speech recognition
  • Parse natural language commands
  • Convert commands to robot actions

Path Planning and Navigation​

  • Use Nav2 for autonomous navigation
  • Implement obstacle avoidance
  • Dynamic replanning

Object Detection and Manipulation​

  • Computer vision for object identification
  • Inverse kinematics for reaching
  • Grasp planning and execution

Full System Integration​

  • Combine all modules into one system
  • Handle edge cases and failures
  • Real-time performance

Evaluation Criteria​

  • System completeness
  • Robustness and reliability
  • Code quality and documentation
  • Demonstration of autonomous task completion

Deliverables​

  • Source code repository
  • System architecture documentation
  • Video demonstration
  • Final presentation
